# 文本处理工具 - Text Tools

一个功能丰富的在线文本处理工具集合，采用拟态（Neumorphic）UI设计，提供直观优雅的用户体验。

## 🌟 功能特性

### 📝 文本处理工具
- **转义/去转义**: 处理字符串中的特殊字符，如换行符、引号、反斜杠等
- **JSON格式化**: 格式化、压缩和验证JSON数据
- **多行文本处理**: 将多行文本转换为引号列表、逗号分隔、数组格式等
- **Base64编码**: Base64编码和解码
- **URL编码**: URL编码和解码
- **哈希计算**: 计算MD5、SHA1、SHA256哈希值

### 🎨 设计特色
- **拟态UI设计**: 柔和的凸起质感，仿佛从背景表面"挤出"
- **微妙阴影效果**: 通过明暗双重阴影营造真实可触的物理感
- **单色系配色**: 采用柔和的浅灰色调，整体和谐统一
- **无硬边框**: 所有元素都呈现圆润的边角
- **交互反馈**: 按钮具有柔和的"被按下"效果

### ⚡ 用户体验
- **响应式设计**: 完美适配桌面和移动设备
- **键盘快捷键**: 支持Ctrl+Enter执行、Ctrl+L清空等快捷操作
- **一键复制**: 快速复制处理结果到剪贴板
- **自动保存**: 输入内容自动保存到本地存储
- **实时状态反馈**: 操作结果实时显示

## 📖 使用说明

### 转义/去转义工具
- **转义**: 将特殊字符转换为转义序列（如 `\n`、`\"`、`\\` 等）
- **去转义**: 将转义序列还原为原始字符
- 支持的转义字符：换行符、回车符、制表符、引号、反斜杠等

### JSON格式化工具
- **格式化**: 将压缩的JSON转换为易读的格式
- **压缩**: 移除JSON中的空白字符，减小文件大小
- **验证**: 检查JSON语法是否正确

### 多行文本处理
- **转为引号列表**: `aaa\nbbb\nccc` → `'aaa',\n'bbb',\n'ccc'`
- **转为逗号分隔**: `aaa\nbbb\nccc` → `aaa, bbb, ccc`
- **转为数组**: `aaa\nbbb\nccc` → `["aaa", "bbb", "ccc"]`
- **去重**: 移除重复的行

### Base64编码工具
- 支持UTF-8文本的Base64编码和解码
- 自动处理中文等多字节字符

### URL编码工具
- URL编码：将特殊字符转换为%编码格式
- URL解码：将%编码还原为原始字符

### 哈希计算工具
- **MD5**: 计算文本的MD5哈希值
- **SHA1**: 计算文本的SHA1哈希值
- **SHA256**: 计算文本的SHA256哈希值

## ⌨️ 键盘快捷键

- `Ctrl + Enter`: 执行当前工具的主要功能
- `Ctrl + L`: 清空当前工具的输入和输出
- `Tab`: 在输入框间切换焦点

## 🎯 技术特点

- **纯前端实现**: 无需服务器，所有处理都在浏览器中完成
- **现代Web技术**: 使用ES6+、Web Crypto API等现代技术
- **安全性**: 所有数据处理都在本地进行，不会上传到服务器
- **兼容性**: 支持现代浏览器（Chrome、Firefox、Safari、Edge）

## 🔧 自定义配置

### 修改主题色彩
在 `styles.css` 中修改CSS变量：
```css
:root {
    --bg-color: #e0e0e0;        /* 背景色 */
    --text-color: #333;         /* 主文本色 */
    --accent-color: #4a90e2;    /* 强调色 */
    /* ... 其他颜色变量 */
}
```

### 添加新工具
1. 在 `index.html` 中添加导航按钮和工具面板
2. 在 `script.js` 中实现相应的处理函数
3. 在 `styles.css` 中添加必要的样式
